# Internship Program Overview & Setup Guide

## 📚 Internship Structure

The internship spans **8 weeks**, each focused on a specific skillset:

### Week 1 – Onboarding and Foundations
- Setup tools and tech stack
- Learn programming basics
- Build a personal portfolio website using **HTML, CSS, and JavaScript**

### Week 2 – Problem-Solving and Algorithms
- Explore sorting/searching algorithms
- Understand **time and space complexity** with Big-O notation

### Week 3 – Web Development Basics
- Dive into **front-end** (HTML, CSS, JS) and **back-end** development
- Create a basic server using **Node.js** or **Flask**

### Week 4 – Intermediate Development
- Learn **Git branching & merging**
- Write **unit tests**
- Build complex projects like an **e-commerce site**

### Week 5 – Deployment & Emerging Technologies
- Deploy projects using **Netlify** or **Heroku**
- Intro to **AI/ML** and **blockchain** integration

### Week 6 – Software Capstone Project Ideas
- Develop functional apps: task managers, chat apps, etc.
- Learn the full development cycle: brainstorming to debugging

### Week 7 – Space Tech Capstone Project Ideas
- Use **NASA, ISRO, and SpaceX APIs**
- Build projects on:
  - Weather monitoring
  - NDVI analysis
  - Navigation systems

### Week 8 – Final Capstone Project & Showcase
- Build a comprehensive final project
- Perform final testing, deployment, and live demo/presentation
